# FitMonk Image Requirements

The FitMonk website requires the following images:

## Logo

1. **fitmonk-logo.png** - Circular logo with monk lifting weights
   - Used in: Logo component throughout the site
   - Recommended size: 200x200px (circular)
   - Location: `/public/images/fitmonk-logo.png`

## Main Images

1. **fitness-workout.jpg** - Image of a person working out in the gym
   - Used in: HomeHero component
   - Recommended size: 1200x800px
   - Location: `/public/images/fitness-workout.jpg`

2. **fitness-app-mobile.jpg** - Image of a fitness app on mobile
   - Used in: App Screenshot section on homepage
   - Recommended size: 800x1600px
   - Location: `/public/images/fitness-app-mobile.jpg`

## User Profile Images

1. **user1.jpg**, **user2.jpg**, **user3.jpg** - Testimonial user profile pictures
   - Used in: Testimonials section on homepage
   - Recommended size: 100x100px (square)
   - Location: `/public/images/user1.jpg`, etc.

Please add these images to the appropriate locations for the website to display correctly.
